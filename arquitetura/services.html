Camada de Servicos ​

A camada de servicos (app/services/) contem a logica de negocio do Kubex, separada dos endpoints (controllers) e dos motores (engines). Cada servico encapsula uma responsabilidade especifica.

Visao Geral ​

Servicos ​

SearchService ​

Arquivo: app/services/search_service.pyResponsabilidade: Orquestra o fluxo de busca: upload → hash → job → API search → resultados.

MetodoDescricao
register_content()Registra imagem: calcula pHash/dHash/aHash, salva ContentHash
create_search_job()Cria job de busca com validacao de plano
get_search_results()Retorna resultados paginados com score

Dependencias: SQLAlchemy, phash_engine, ContentHash model

ImageProcessor ​

Arquivo: app/services/image_processor.pyResponsabilidade: [LGPD] Processamento de imagem em memoria — nunca persiste bytes.

MetodoDescricao
compute_phash()Calcula perceptual hash (pHash + dHash + aHash)
phash_distance()Distancia Hamming entre dois hashes
get_faces_from_image()Extrai faces para FaceID (face_recognition)

Dependencias: PIL, imagehash, httpx

IdentityService ​

Arquivo: app/services/identity_service.pyResponsabilidade: [LGPD] Verificacao de identidade facial. Selfie → embedding → match. Status: none → pending → verified/rejected.

MetodoDescricao
get_identity_status()Status atual da verificacao do usuario
submit_identity()Processa selfie e cria embedding (nunca armazenado em disco)

Dependencias: SQLAlchemy, CreatorEmbedding model, EncryptionService

EncryptionService ​

Arquivo: app/services/encryption_service.pyResponsabilidade: [LGPD] Criptografia per-user de embeddings faciais via AES-256-GCM com PBKDF2.

MetodoDescricao
encrypt_embedding()Criptografa embedding 512-d com chave derivada do user_id
decrypt_embedding()Decripta embedding (impossivel em batch — chave por usuario)

Dependencias: cryptography (AESGCM), ENCRYPTION_KEY

TakedownService ​

Arquivo: app/services/takedown_service.pyResponsabilidade: Fluxo completo de takedown: validacao de propriedade → criacao de registro → coleta de evidencia → envio.

MetodoDescricao
initiate_takedown()Valida ownership, cria Takedown, enfileira task Celery
list_takedowns()Lista takedowns do usuario com filtros
get_takedown_details()Detalhes completos com historico de verificacoes

Dependencias: SQLAlchemy, EvidenceService, takedown_engine, Celery

EvidenceService ​

Arquivo: app/services/evidence_service.pyResponsabilidade: Coleta de evidencia para takedowns. Modo dual: httpx (dev, GET + HTML hash) → Playwright (prod, screenshot).

MetodoDescricao
collect_evidence()Captura pagina (HTML hash ou screenshot), salva no storage
_is_safe_url()Guarda SSRF: bloqueia RFC-1918, loopback, IMDS, non-HTTP(S)
extract_domain()Extrai dominio limpo da URL

Dependencias: httpx, Playwright (prod), StorageService, DNS resolution

VerificationService ​

Arquivo: app/services/verification_service.pyResponsabilidade: Multi-verificacao de remocao apos takedown. 3 metodos independentes com votacao 2-de-3.

MetodoDescricao
verify_removal()Executa 3 verificacoes e retorna resultado por votacao
_check_http_status()404/410/451 = removido
_check_content_removal()Busca indicadores de remocao no HTML
_check_phash_comparison()Compara pHash antes/depois

Dependencias: httpx, imagehash, user-agent rotation

StorageService ​

Arquivo: app/services/storage_service.pyResponsabilidade: Abstrai armazenamento. Dev: filesystem local (/tmp/kubex-evidence). Prod: Cloudflare R2 (S3-compatible).

MetodoDescricao
upload_evidence()Upload de arquivo com content-type automatico
download_evidence()Download de evidencia por key
_get_s3_client()Client boto3 singleton para R2

Dependencias: boto3 (S3), R2_ACCOUNT_ID, R2_ACCESS_KEY_ID, R2_SECRET_ACCESS_KEY

AsaasService ​

Arquivo: app/services/asaas_service.pyResponsabilidade: Client da API Asaas para pagamentos. Sem API key: mock mode (logs no console). Com key: processamento real.

MetodoDescricao
create_customer()Cria customer no Asaas (email + name)
create_subscription()Cria assinatura recorrente (PIX, cartao, boleto)
create_charge()Cria cobranca avulsa (pay-per-search)
list_invoices()Lista faturas do customer

Dependencias: httpx, Asaas API (sandbox ou producao), ASAAS_API_KEY

EmailService ​

Arquivo: app/services/email_service.pyResponsabilidade: Envio de emails. Dev: loga link no console. Prod: Resend API com templates HTML.

MetodoDescricao
send_verification_email()Email de verificacao de conta
send_password_reset()Email de reset de senha com link temporario

Dependencias: httpx, Resend API, RESEND_API_KEY

HibpService ​

Arquivo: app/services/hibp_service.pyResponsabilidade: Client HaveIBeenPwned API v3. Dev: mock data. Prod: Redis cache 1h.

MetodoDescricao
check_password()k-anonymity password check (SHA-1 prefix)
check_email()Verifica vazamentos por email
get_breaches()Lista breaches detalhados

Dependencias: httpx, Redis (cache), HIBP_API_KEY

NameSearchService ​

Arquivo: app/services/name_search_service.pyResponsabilidade: Servico DB para BrandWatch (busca por nome). Gerencia jobs, targets e perfis legitimos.

MetodoDescricao
create_name_search_job()Cria job de busca por nome com variacoes
get_active_targets()Lista targets ativos (globais + do usuario)
get_targets_by_ids()Busca targets por IDs especificos

Dependencias: SQLAlchemy, CrawlTarget model, UserProfile model

SourceScoringService ​

Arquivo: app/services/source_scoring_service.pyResponsabilidade: SourceIntel — pontua fontes descobertas (0.0-1.0) baseado em indicadores de pirataria.

MetodoDescricao
score_source()Pontua fonte: keywords, adult markers, cross-refs, TLD, uploads
_check_domain_keywords()Verifica keywords no dominio (piracy, leak, etc.)
_check_content_keywords()Verifica keywords no conteudo da pagina

Dependencias: regex, urllib

SourceSeedService ​

Arquivo: app/services/source_seed_service.pyResponsabilidade: SourceIntel — importa seeds JSON → DiscoveredSource. Promove fontes aprovadas para tabelas de monitoramento.

MetodoDescricao
import_seeds_from_json()Upsert de seeds por URL
promote_source()Promove fonte aprovada → WebCrawlerTarget/MonitoredTelegramChannel/etc.
auto_provision()Provisiona automaticamente fontes acima do threshold

Dependencias: SQLAlchemy, DiscoveredSource model, JSON seeds

ReportPdfService ​

Arquivo: app/services/report_pdf_service.pyResponsabilidade: Geracao de relatorios PDF profissionais (face search + monitoramento).

FaceOptoutCache ​

Arquivo: app/services/face_optout_cache.pyResponsabilidade: Cache Redis de opt-out embeddings (elimina round-trip Milvus).

FaceBatchInsert ​

Arquivo: app/services/face_batch_insert.pyResponsabilidade: Buffer de batch insert para Milvus (reduce gRPC round-trips).

KitBoService ​

Arquivo: app/services/kit_bo_service.pyResponsabilidade: Geracao de Kit B.O. PDF para delegacias.
