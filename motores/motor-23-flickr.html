Visao Geral ​

O Flickr Monitor busca fotos publicas no Flickr por username, tags ou texto livre correspondente ao nome/marca do criador. Usa a API publica do Flickr via httpx (sem API key necessaria para busca basica) — mais leve e confiavel que o Facebook Monitor por nao exigir renderizacao de browser.

Caracteristicas:

  • API publica Flickr REST (flickr.photos.search) — sem API key
  • Paginacao com MAX_PAGES=5, PHOTOS_PER_PAGE=50 (250 fotos max por scan)
  • Constroi URLs de thumbnail via _build_photo_url() padrao Flickr CDN
  • Circuit breaker com 5 falhas → janela de 5 minutos aberta
  • Mock mode automatico quando FLICKR_MONITOR_ENABLED=false

Engine: FlickrMonitor ​

python
# app/engines/flickr_monitor.py

MAX_PAGES = 5
PHOTOS_PER_PAGE = 50  # max 250 fotos por scan
FLICKR_API = "https://www.flickr.com/services/rest/"

def _build_photo_url(
    farm: int, server: str, photo_id: str, secret: str, size: str = "q"
) -> str:
    """
    Constroi URL de thumbnail Flickr CDN.
    size='q' = 150x150 (suficiente para pHash)
    """
    return f"https://farm{farm}.staticflickr.com/{server}/{photo_id}_{secret}_{size}.jpg"


class FlickrMonitor:
    def __init__(self):
        self.circuit = flickr_circuit  # breaker: 5 fails → 5min open

    async def search_by_username(
        self, username: str, page: int = 1
    ) -> list[str]:
        """
        Busca fotos publicas por texto (username/nome).
        Retorna lista de URLs de thumbnail.
        """
        if not settings.FLICKR_MONITOR_ENABLED:
            return []  # mock mode

        async with self.circuit:
            urls = []
            for p in range(1, MAX_PAGES + 1):
                resp = await httpx.get(FLICKR_API, params={
                    "method": "flickr.photos.search",
                    "format": "json",
                    "nojsoncallback": 1,
                    "text": username,
                    "per_page": PHOTOS_PER_PAGE,
                    "page": p,
                })
                data = resp.json()
                photos = data.get("photos", {}).get("photo", [])
                if not photos:
                    break
                for ph in photos:
                    url = _build_photo_url(
                        ph["farm"], ph["server"], ph["id"], ph["secret"]
                    )
                    urls.append(url)
            return urls

Relacoes ​

  • PIB: Toda imagem encontrada e indexada via index_found_image() em platform_intelligence.py
  • Fingerprint: pHash compara thumbnails baixados com hashes do usuario
  • FaceID: Face recognition opcional no pipeline de comparacao
  • Whitelist: Dominios na whitelist_domains sao ignorados no PIB

Seguranca & LGPD ​

  • Nao armazena imagens — thumbnails processados em memoria e descartados apos pHash
  • API publica Flickr (sem API key para busca basica)
  • Circuit breaker protege contra cascata de falhas (5 falhas → 5min open)
  • PII nunca em logs — apenas IDs de foto e query de busca
  • Mock mode quando FLICKR_MONITOR_ENABLED=false
