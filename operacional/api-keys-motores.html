🔑 Motores que Requerem API Keys (19 motores) ​

GRÁTIS / PAY-AS-YOU-GO ​

1. SerpAPI (Motor 1: Lens)

  • Custo: 100 buscas/mês grátis → $0.002 por busca (Google), $0.005 (Bing)
  • Como obter:
    1. Acessar: https://serpapi.com/users/sign_up
    2. Criar conta com email
    3. Copiar API Key da dashboard (Settings → API Key)
    4. Adicionar em .env: SERPAPI_API_KEY=xxx
  • Limite de teste: 100 buscas/mês
  • Quota anual (estimada): R$1-2 mil para 1M buscas

2. Telegram API (Motor 5: Telegram Monitor)

  • Custo: GRÁTIS (chamadas ilimitadas)
  • Como obter:
    1. Conversar com @BotFather no Telegram
    2. Criar novo bot: /newbot
    3. Copiar token: BOT_TOKEN
    4. Adicionar em .env: TELEGRAM_BOT_TOKEN=xxx:yyy
    5. Para client API: phone number + 2FA code (tedioso)
  • Nota: Client API requer login com conta pessoal Telegram
  • Alternativa: Usar apenas Bot API (mais simples)

3. Discord API (Motor 9: Discord Monitor)

  • Custo: GRÁTIS (webhooks ilimitados)
  • Como obter:
    1. Discord Developer Portal: https://discord.com/developers/applications
    2. New Application → Copiar Token
    3. Em "Bot", gerar/copiar token
    4. Adicionar em .env: DISCORD_BOT_TOKEN=xxx
  • Requere: Bot deve estar em servidor de teste
  • Limite: Rate limiting: 50 req/segundo

4. Reddit API (Motor 13: Reddit Monitor)

  • Custo: GRÁTIS (100 requests/minute)
  • Como obter:
    1. Registrar em: https://www.reddit.com/register
    2. Ir para: https://www.reddit.com/prefs/apps (bottom)
    3. Create app → choose "script"
    4. Notar: client_id (abaixo do nome), client_secret
    5. .env: REDDIT_CLIENT_ID=xxx, REDDIT_CLIENT_SECRET=yyy
  • Rate limit: 60 requests/minute (user-agent específico)
  • Requere: Reddit account + Reddit app registration

5. Twitter/X API (Motor 15: X Monitor)

  • Custo: Grátis (Essential tier — 450k tweets/mês)
  • Como obter:
    1. Developer Portal: https://developer.twitter.com/en/portal/dashboard
    2. Projects & Apps → Create app
    3. Pegar: API Key, API Secret, Bearer Token
    4. .env: TWITTER_API_KEY=xxx, TWITTER_API_SECRET=yyy, TWITTER_BEARER_TOKEN=zzz
  • Essential tier: 450k tweets/month, 450 req/15min
  • Paid: Elevated+ (2M tweets/mês, $100/mês)
  • Nota: Conta X necessária + aprovação da aplicação (pode levar 48h)

6. Google Alerts (Motor 14: AlertsFeed)

  • Custo: GRÁTIS
  • Como obter:
    1. Google Alerts: https://alerts.google.com
    2. Criar alertas manualmente
    3. Endpoint admin da Kubex exporta esses alertas
  • Nota: Requer login Gmail + configuração em UI

7. Flickr API (Motor 23: Flickr Monitor)

  • Custo: GRÁTIS (sem quotas documentadas)
  • Como obter:
    1. https://www.flickr.com/services/api/misc.api_keys.html
    2. Apply for an API key → Non-commercial
    3. Copiar API Key + Secret
    4. .env: FLICKR_API_KEY=xxx, FLICKR_API_SECRET=yyy
  • Rate limit: Não documentado, parece ilimitado

8. Instagram Graph API (Motor 18: Instagram Monitor)

  • Custo: GRÁTIS até limites razoáveis
  • Como obter:
    1. Meta for Developers: https://developers.facebook.com
    2. Create app → Business
    3. Add Instagram Graph API product
    4. Gerar access token (User Token)
    5. .env: INSTAGRAM_ACCESS_TOKEN=xxx
  • Requer: Facebook/Meta Business Account
  • Rate limit: 200 requests/hour

9. DuckDuckGo Search (descontinuado)

  • ❌ Removido em Sprint F (Motor 1 agora usa SerpAPI apenas)

PAGOS / FREEMIUM ​

10. BrandWatch (Motor 4: Social Listening)

  • Custo: ~$5000/mês enterprise
  • Como obter: Enterprise sales → não cobre Kubex MVP
  • Alternativa: Usar Telegram + Twitter + Reddit + Google Alerts

11. Darknet/Dark Web Monitor (Motor 16: DarkWeb Monitor)

  • Custo: Varia ($500-2000/mês depende do serviço)
  • Alternativa: Usar Telegram dark bot lists (grátis)

12. Marketplace Monitors (Motor 19: Marketplace Monitor)

  • Custo: Varia por marketplace
  • Inclusos: Amazon, eBay, Alibaba, Mercado Livre
  • Grátis: Web scraping (com cuidado com ToS)

13. Takedown Services (Motor 7: Takedown)

  • Custo: Varia (DMCA.com ~$50 por takedown)
  • Alternativa: Contato manual com plataformas

INTERNOS / SEMPAI (Não requerem configuração) ​

14. OCR (Motor 20): Local ONNX models ✅ 15. VideoPrint (Motor 11): Local ffmpeg ✅ 16. Orchestrator (Motor 8): Interno ✅ 17. Sentinel (Motor 27): Health monitor interno ✅ 18. ShieldAI (Motor 25): Local model ✅ 19. AutoDiscovery (Motor 24): Crawler + análise interna ✅ 20. SourceIntel (Motor 17): Database interna ✅

  3. Testar com /api/v1/test/motor/{motor_id}

Segurança ​

⚠️ NÃO commitir API keys no git!

  • Usar .env (gitignored)
  • Em produção: usar secrets manager (AWS Secrets Manager, HashiCorp Vault)
  • Rotacionar keys regularmente
  • Usar rate limiting para evitar vazamento
